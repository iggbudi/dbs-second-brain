## Auto-Sync

Knowledge di-sync otomatis saat Pi session berakhir via custom extension.

### Manual Sync
```bash
# Dari server
./scripts/sync-knowledge.sh

# Atau langsung
cd /var/www/lk.pjdigital.top
engram sync --all
engram export ~/lk-ukms-knowledge/memories/engram-export.json
cd ~/lk-ukms-knowledge
git add -A && git commit -m "manual sync" && git push
```

### Engram Memory Import (di machine lain)
```bash
cd ~/lk-ukms-knowledge
engram import memories/engram-export.json
```
